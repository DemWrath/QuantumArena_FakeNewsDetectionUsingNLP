**Milestone 2 Goal**: Implement deep semantic analysis and integrate third-party source reliability signals to move beyond 2020-era static datasets.

## Phase 3: Transformer-based NLP Analysis
**Goal**: Upgrade the core classification engine from static ML to contextual DistilBERT and runtime LLM assessment.
- **NLP-01**: Setup `transformers` library and integrate a pre-trained DistilBERT model for modern news style analysis.
- **NLP-02**: Implement the "Emotional Tone" detector using Gemini PRO inference at runtime.
- **NLP-03**: Logic to compare Headline semantics vs. Body semantics for clickbait flagging.

## Phase 4: Source Intelligence & Live Fact-Scraping
**Goal**: Move beyond the text of the article to analyze the reputation and history of the publisher.
- **SRC-01**: Build the live scraper for AllSides.com bias ratings.
- **SRC-02**: Build the live scraper for MediaBiasFactCheck.com reliability scores.
- **SRC-03**: Metadata collection system for Domain Age (WHOIS) and SSL validity signals.

## Phase 5: Layer Fusion & Pipeline 2.0
**Goal**: Unify all new intelligence layers into the master `pipeline.py` and optimize for performance.
- Fuse DistilBERT and Source ratings into the unified JSON diagnostic report.
- Optimize scraper caching to maintain user performance levels.
- Perform end-to-end integration testing for Milestone 2 requirements.

## Phase 6: Frontend Data Restructuring
**Goal**: Restructure the static `index.html` UI mockup to dynamically accept and display the precise data models generated by the backend pipeline.
- Implement URL intake blocks for scraper initiation.
- Mount dynamic DOM IDs for JSON payload extraction.
- Create visual indicators for GenAI Emotion outputs (Fear, Outrage, Sensationalism).
- Replace generic credibility numbers with actual Scraper logic structures (AllSides, MBFC).
