# Requirements: Media Analysis Framework

**Defined:** 2026-04-17
**Core Value:** Combat misinformation via transparent, multi-layer intelligence.

## Milestone 1 Requirements (v1.0)
- [x] **REQ-01**: Orchestration pipeline bridging ML & Bias layers. (Verified)
- [x] **REQ-02**: URL scraping integration using `newspaper3k`. (Verified)
- [x] **REQ-03**: Semantic claim and evidence extraction. (Verified)

## Milestone 2 Requirements (v2.0)

### Layer 2: Semantic Analysis (NLP)
- [x] **NLP-01**: Integrate `DistilBERT` transformer for style and propaganda detection.
- [x] **NLP-02**: Implement runtime LLM (Gemini) inference for emotion/outrage scoring.
- [x] **NLP-03**: Clickbait detection via headline vs. body content consistency check.

### Layer 3: Source Credibility (SRC)
- [x] **SRC-01**: Automated live scraper for **AllSides** bias ratings.
- [x] **SRC-02**: Automated live scraper for **MediaBiasFactCheck** reliability scores.
- [x] **SRC-03**: Fallback inference and reputation scraping mechanism for unindexed domains.

## Future Requirements (v3.0+)

- [ ] **EXPL-02**: BERT attention heatmap injection for UI. (Layer 4)
- [ ] **OUT-01**: JSON/PDF report export functionality. (Layer 5)
